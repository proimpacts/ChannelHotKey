# Channel Hotkeys

Channel Hotkeys is a custom Vencord plugin that lets you create hotkeys to quickly jump to Discord channels, DMs, and voice channels.

Use shortcuts like **Control + G + H** to instantly join a voice channel or **Control + S + D** to open a chat without having to search for it.

This plugin is not officially supported by Vencord and requires a custom plugin setup.

For installation, follow this tutorial:
https://youtu.be/XmVNRKrphlw?si=7Ml5c3yyK5tl1YW3

Keep in mind that custom plugins may take a little extra setup.
